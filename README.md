üéØ Business Objectives

The analysis aims to answer the following business questions:

How does the number of bedrooms impact pricing?

Which properties are the most expensive?

Which areas offer the best value based on price per square foot?

üìà Key Insights

Property prices vary significantly across London locations.

Property size (area) is one of the strongest predictors of price.

Houses generally command higher prices than flats.

Price per square foot reveals better value areas that are not obvious from total price alone.

Bedrooms and area have a strong positive relationship with price.
